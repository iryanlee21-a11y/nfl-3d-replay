Visualize a real NFL play in 3D and watch it from player POVs (QB, RB, etc.) using Unity.

## Features

- 3D field playback in Unity (URP)
- Head-mounted POV cameras per player
- Replay controls & player switching

## Setup
1. Open with **Unity (URP)**.
2. Scene: `Assets/Scenes/MainScene.unity`
3. Prefab: `Assets/Prefabs/Player_dummy.prefab`
